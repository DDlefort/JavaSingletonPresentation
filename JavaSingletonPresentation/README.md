# Java Singleton Presentation

This repository contains simple Java examples for explaining the Singleton design pattern.

## Topic
A Singleton is a design pattern that ensures a class has only one object and provides a way to access that same object throughout the program.

## Files

### Singleton-Version
This version uses the Singleton pattern.
- Only one `School` object is created
- The same object is returned every time
- Final comparison result is `true`

### Non-Singleton-Version
This version does not use the Singleton pattern.
- A new `School` object is created each time
- Different objects are returned
- Final comparison result is `false`

## School name used
Tulsa Community College

## Purpose
These examples were created for a class presentation to compare normal object creation with Singleton object creation in Java.
